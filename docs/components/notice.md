### CountUp Props
| 属性 | 说明 | 类型 | 默认值 |
| :- | :- | :- | :- |
| icon | 左侧图标(暂不可用) | string | -- |
| text | 通知文本 | string | --- |
| speed | 速度 | number | 50 |
| delay | 延迟(s) | number | 1 |

| 事件名称 | 说明 | 回调参数 |
| :- | :- | :- |
